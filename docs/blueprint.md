# **App Name**: DesignVerse

## Core Features:

- Designer Profiles: Detailed profiles showcasing designers' portfolios, skills, and client reviews.
- Project Request Form: A structured form for clients to submit detailed project briefs including style preferences, budget, and deadlines.
- Designer Marketplace: A searchable marketplace with filters for service category, budget, experience level, and availability.
- Direct Messaging: A real-time chat interface enabling direct communication between clients and designers.
- Personalized Designer Recommendations: AI-powered recommendations that match clients with designers based on project requirements, style preferences, and past successful collaborations; incorporates tool use to adaptively use client and designer information.

## Style Guidelines:

- Primary color: Vibrant teal (#008080) for a sense of creativity and trustworthiness.
- Background color: Light gray (#F0F0F0) for a clean and modern feel.
- Accent color: Light blue (#ADD8E6) to highlight key interactive elements.
- Font pairing: 'Poppins' (sans-serif) for headings, 'PT Sans' (sans-serif) for body text. Note: currently only Google Fonts are supported.
- Use clear and professional icons for categories and actions, with a style matching the modern aesthetic.
- Responsive grid layout adapting to desktop, tablet, and mobile, ensuring optimal viewing experience on all devices.
- Subtle animations to enhance user interaction, such as hover effects on designer cards and smooth transitions between pages.
